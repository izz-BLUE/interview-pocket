# Interview Pocket 路线图

## 已完成

### V0.1 - 基础功能

- Markdown 文件导入
- 题库列表展示
- 关键词搜索
- 普通卡片复习
- 自评打分
- 掌握度计算

### V0.2 - 功能完善

- 错题复习模式
- 面试前突击模式
- 来源筛选
- 导入报告
- 复习统计增强
- 稳定性清理
- 大题库加载优化
- 题库来源删除
- 复习状态筛选

## 计划中

### P0 - 高优先级

- README 截图补充
- 单题编辑功能
- 数据导出（CSV / JSON）

### P1 - 中优先级

- 更强的 Markdown 解析器（支持更多格式）
- 分页 / 虚拟列表（支持更大题库）
- 自动化测试（IPC 层单元测试）
- 复习算法优化（更精确的间隔重复）

### P2 - 低优先级

- 可选 AI 题目分析
- 多题库合并
- 更多统计维度

## 不计划做

- 用户系统（登录 / 注册）
- 云同步
- 多人协作
- 商业化 SaaS
- 自动发布安装包
