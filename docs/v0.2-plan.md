# V0.2 Plan

## V0.2 目标

从"能导入和复习"升级为"更像面试训练工具"。

核心方向：
- 强化薄弱环节：错题复习、突击模式
- 提升数据可见性：掌握度统计、复习记录
- 优化导入体验：导入报告增强、来源筛选

## V0.2 不做什么

明确不做：

- **AI 接入**：不接入任何 LLM，保持纯本地。
- **云同步**：数据不上云，保持离线。
- **多用户系统**：单机单用户，不做账号。
- **复杂权限**：不做角色和权限管理。
- **Windows exe 打包**：暂不处理打包和签名。
- **系统托盘**：暂不实现托盘常驻。
- **全局快捷键**：暂不实现全局热键。
- **题库市场**：不做题库分享和下载。
- **复杂 SM-2 算法**：保持当前简化版间隔复习。

## V0.2 功能范围

### P0 - 核心训练功能

#### 1. 错题复习模式

**目标**：针对答错的题目进行专项复习。

**功能**：
- 从题库列表进入"错题复习"
- 只展示 `wrong_count > 0` 的题目
- 按 `wrong_count` 降序排列（错得最多排最前）
- 支持错题清零（掌握后重置 wrong_count）

**数据来源**：`review_progress.wrong_count`

#### 2. 面试前突击模式

**目标**：面试前快速过一遍重点题目。

**功能**：
- 从首页进入"突击模式"
- 选择题库来源（source_file）
- 快速浏览模式：只看题目 → 想答案 → 看要点 → 下一题
- 不记录复习数据，纯浏览
- 支持标记"不确定"题目，最后统一复习

#### 3. 题库来源筛选

**目标**：按来源文件筛选题目。

**功能**：
- 题库列表增加"来源"筛选下拉
- 列出所有 `source_file` 去重值
- 选择后只展示该来源的题目
- 支持"全部"选项

**数据来源**：`questions.source_file`

#### 4. 导入报告增强

**目标**：导入后展示更详细的解析结果。

**功能**：
- 导入成功后展示报告页
- 显示：新增题目数、更新题目数、跳过题目数
- 显示：解析到的答案数、追问点数、记忆锚点数
- 显示：解析失败的题目列表（如有）
- 支持"查看导入的题目"快捷跳转

### P1 - 数据可见性

#### 5. 基础标签展示

**目标**：展示题目的分类标签。

**功能**：
- 题库列表显示 `category` 标签
- 题目详情页显示分类标签
- 支持按分类筛选（复用 source_file 筛选组件）

**数据来源**：`questions.category`

#### 6. 复习记录详情

**目标**：查看单题的复习历史。

**功能**：
- 题目详情页增加"复习记录"tab
- 展示该题的所有 `review_records`
- 显示：复习时间、自评分数
- 按时间倒序排列

**数据来源**：`review_records`

#### 7. 简单掌握度统计

**目标**：直观展示掌握程度。

**功能**：
- 题库列表显示掌握度进度条
- 掌握度 = `mastery_score`（0-100）
- 颜色映射：0-40 红色、40-70 黄色、70-100 绿色
- 首页统计：平均掌握度、低掌握度题目数

**数据来源**：`review_progress.mastery_score`

#### 8. 今日复习完成页优化

**目标**：复习完成后展示更有价值的信息。

**功能**：
- 完成页展示本次复习统计
- 显示：复习题数、平均分、最高分、最低分
- 显示：新掌握题目数（mastery_score 提升的）
- 显示：仍需加强题目数（wrong_count > 0）
- 提供"查看错题"快捷入口

### P2 - 体验优化

#### 9. 题目详情 UI 优化

**目标**：优化答案展示体验。

**功能**：
- 答案区域支持折叠/展开
- 记忆锚点高亮显示
- 追问点以折叠列表展示
- 支持答案区域滚动位置记忆

#### 10. 题库空状态优化

**目标**：无题目时展示引导。

**功能**：
- 题库为空时展示引导页
- 提供"导入题库"快捷按钮
- 展示支持的 Markdown 格式示例
- 提供示例文档下载（可选）

#### 11. 搜索结果筛选增强

**目标**：搜索结果更精准。

**功能**：
- 搜索结果高亮匹配关键词
- 搜索支持按来源、分类筛选
- 搜索结果数量显示
- 无结果时展示"未找到"提示

## 数据模型影响

V0.2 使用现有字段，暂不新增复杂表：

| 字段 | 用途 |
|------|------|
| `review_progress.wrong_count` | 错题复习筛选 |
| `review_progress.mastery_score` | 掌握度统计 |
| `questions.source_file` | 来源筛选 |
| `questions.category` | 分类标签 |
| `questions.memory_point` | 记忆锚点展示 |
| `review_records` | 复习记录详情 |

可能新增：
- `import_logs` 表：记录导入历史（用于导入报告增强）

## 技术风险

| 风险 | 影响 | 应对 |
|------|------|------|
| sql.js 内存限制 | 大量复习记录查询可能慢 | 分页查询，限制单次返回数量 |
| source_file 数据不一致 | 同一来源文件名不同 | 导入时规范化文件名 |
| 突击模式不记录数据 | 用户可能误以为在复习 | UI 明确提示"浏览模式" |
