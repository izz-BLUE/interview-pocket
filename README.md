# Interview Pocket

Windows 本地面试复习训练客户端。

## 项目定位

基于本地 Markdown 面试文档，将面试手册解析为结构化题库，支持离线搜索、答案展示、复习打分、今日待复习队列和复习进度统计。

## 项目背景

面试准备过程中常见问题：

- **面试题分散**：面试题散落在多个 Markdown 文档中，难以快速定位和系统复习。
- **只能阅读不能训练**：普通文档阅读没有反馈，无法衡量掌握程度。
- **缺少复习节奏**：没有间隔复习机制，容易遗忘。
- **数据安全顾虑**：不想把面试资料上传到云端或交给 AI 处理。

Interview Pocket 解决这些问题：**本地运行、离线可用、数据不上传、不依赖 AI**。

## 核心功能

V0.1 已实现功能：

- **Markdown 导入**：将本地 Markdown 面试文档导入为结构化题库。
- **结构化解析**：自动识别题目编号、标准答案、简短回答、记忆锚点、追问点等。
- **答案展示**：查看完整答案、简短回答、记忆锚点、追问点。
- **题库列表**：浏览所有已导入题目。
- **关键词搜索**：按关键词搜索题目，支持相关性排序。
- **复习统计**：今日待复习、今日已复习、总题数统计。
- **卡片式复习**：逐题复习，先看题目思考，再查看答案。
- **自评打分**：1-5 分自评，系统根据评分安排下次复习时间。
- **连续下一题**：完成一题后直接进入下一题，保持复习节奏。
- **本地持久化**：SQLite 数据库保存所有数据。
- **数据库迁移**：支持版本升级时自动迁移数据库结构。

## 本地数据说明

数据保存位置：

```
%APPDATA%/interview-pocket/interview-pocket.db
```

- **不上传云端**：所有数据保存在本地 SQLite 文件。
- **不依赖 AI**：纯本地运行，无需网络。
- **不依赖后端**：无服务器，无账号系统。

## Markdown 解析说明

当前支持的 Markdown 格式：

**题目识别**：
- `P0-1`、`Q1`、`J1`、`B1`、`1.1` 等编号格式

**答案识别**：
- 推荐回答 / 参考回答
- 简短回答
- 深入回答

**辅助信息**：
- 一句话锚点 / 核心句
- 追问点 / 可能追问
- 注意 / 禁忌说法

**代码块**：
- `text` 代码块作为普通答案正文保留
